# 文件拆分邮件发送工具 v0.1 需求说明

## 1. 项目目标

在外网电脑上选择一个常用办公文件或压缩包，将其自动拆分为多封符合公司邮件大小限制的附件，并通过网易邮箱发送至指定的公司邮箱。

内网端不安装软件。收件人下载全部分片后，按照邮件正文提供的 Windows 合并命令恢复原文件。

## 2. 实现形式

- 应用类型：跨平台桌面应用。
- 外网发送端系统：Windows、macOS。
- 内网接收端系统：Windows。
- 推荐技术实现：Electron + TypeScript。
- 界面语言：简体中文。

## 3. v0.1 功能范围

### 3.1 文件范围

第一版支持以下文件类型：

- Word：`doc`、`docx`
- Excel：`xls`、`xlsx`
- PowerPoint：`ppt`、`pptx`
- PDF：`pdf`
- 压缩文件：`zip`、`rar`、`7z`

每次发送任务只允许选择一个文件。

### 3.2 发件与收件规则

- 发件邮箱：第一版仅支持网易 163 邮箱。
- 收件人：每次输入一个收件邮箱。
- 不支持抄送、密送或多收件人。
- 同一时间仅允许一个发送任务。
- 用户首次使用时填写网易邮箱地址和客户端授权密码。
- 网易服务器配置由应用预设，用户无需填写 SMTP 地址、端口等技术信息。
- 默认勾选“记住此账号”；授权密码仅加密保存在当前电脑的系统安全凭据中。
- 设置页支持移除已保存账号。

## 4. 文件拆分规则

### 4.1 大小规则

- 公司邮件限制按“单封完整邮件不超过 9.9MB”执行。
- 原始文件分片默认目标大小约为 7MB。
- 应用在发送前按邮件实际编码后的大小进行校验；必要时自动缩小分片，确保邮件整体不超过 9.9MB。
- 文件小于约 7MB 时不拆分，直接作为原文件附件发送。
- 文件超过 200MB 时，应用提示邮件数量较多、可能触发邮箱频率限制；用户再次确认后可继续发送。

### 4.2 拆分方式

- 所有支持的文件均采用原始字节连续切分。
- 不解压压缩包，不重新压缩文件，不转换 Word、Excel、PDF 等文件格式。
- 大文件的单个分片不能直接打开；按顺序合并后应与原文件完全一致。
- 分片附件名称采用固定排序格式，例如原文件 `项目汇总.xlsx` 被拆为 3 份时：

  ```text
  项目汇总.xlsx.part001of003
  项目汇总.xlsx.part002of003
  项目汇总.xlsx.part003of003
  ```

## 5. 邮件生成与发送流程

1. 用户选择文件并输入收件邮箱。
2. 应用完成拆分计算，显示原文件大小、预计单份大小、邮件总数和收件人。
3. 用户点击一次“确认发送”后，应用自动连续发送全部邮件。
4. 邮件之间默认间隔约 2 秒。
5. 每封邮件主题按以下格式生成：

   ```text
   原文件名_Part1
   原文件名_Part2
   ```

6. 每封邮件正文自动包含：

   - 原始文件名和文件大小；
   - 当前分片序号与总分片数；
   - Windows 收件端合并说明；
   - 根据实际分片生成的完整 `copy /b` 合并命令；
   - 原始文件的校验值，用于核验合并后的文件完整性。

## 6. 内网 Windows 合并方式

收件人不需要安装软件，操作方式如下：

1. 将全部分片附件下载到同一个文件夹。
2. 在该文件夹的地址栏输入 `cmd` 并按回车，打开命令提示符。
3. 复制邮件正文中的合并命令并执行。
4. 在该文件夹中得到 `原文件名_合并后.扩展名`。

示例：

```bat
copy /b "项目汇总.xlsx.part001of003"+"项目汇总.xlsx.part002of003"+"项目汇总.xlsx.part003of003" "项目汇总_合并后.xlsx"
```

## 7. 进度、失败与恢复

- 发送界面显示总进度、当前发送的邮件序号和每封邮件的成功/失败状态。
- 某封邮件发送失败时，自动重试 3 次。
- 连续 3 次失败后，暂停整个批次，并显示失败原因。
- 用户可从失败的那一封继续发送，不重复已成功发送的分片。
- 用户可主动暂停发送；当前正在发送的邮件完成后，停止后续邮件。
- 已暂停任务保存本地进度，后续可继续执行。

## 8. 发送历史与隐私

- 本机保留最近 30 天的发送历史。
- 历史记录包含：文件名、文件大小、收件邮箱、发送时间、分片总数和最终状态。
- 不保存原文件副本。
- 历史记录中不保存授权密码。
- 用户可在设置中一键清空历史记录。

## 9. 页面规划

### 9.1 发送页

- 选择文件。
- 输入收件邮箱。
- 显示拆分预览。
- 一键确认发送。

### 9.2 发送进度页

- 显示邮件总数、当前进度和单封结果。
- 支持暂停发送和失败后继续。

### 9.3 历史记录页

- 查看最近 30 天的发送任务和结果。
- 支持清空本地历史。

### 9.4 设置页

- 配置或移除网易账号。
- 管理已保存账号。
- 清空历史记录。

## 10. 分阶段开发目标

### 阶段 1：本地拆分原型

完成文件选择、文件类型校验、邮件大小计算、分片生成、邮件主题和正文生成，以及 Windows 合并命令生成。

**验收目标：** 对本地样例文件完成拆分后，可通过 `copy /b` 合并并与原文件校验值一致。

### 阶段 2：网易自动发送

完成网易授权密码配置、安全保存、发送预览、一键发送、2 秒发送间隔、失败重试、暂停和继续发送。

**验收目标：** 使用真实网易邮箱向公司邮箱成功发送一个多分片文件。

### 阶段 3：可靠性与记录

完成发送进度、30 天历史记录、错误提示、任务恢复和临时文件清理。

**验收目标：** 在断网、应用关闭和发送失败等场景下，任务状态和恢复行为正确。

### 阶段 4：跨平台打包与试用

生成 Windows 和 macOS 安装包，完成常用文件类型、200MB 文件和内网 Windows 合并的完整测试。

**验收目标：** 试用同事能够独立完成“配置邮箱—发送文件—内网合并”的完整流程。

## 11. 暂不纳入 v0.1 的功能

- QQ、企业邮箱及其他 SMTP 邮箱。
- 多文件批量发送。
- 多收件人、抄送、密送和通讯录。
- 内网图形化合并工具。
- 收件邮箱域名白名单。
- 自动更新和正式签名发布。
