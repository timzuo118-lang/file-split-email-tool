# 文件拆分邮件发送工具 v0.1

用于在外网电脑上将单个办公文件拆分为符合邮件大小限制的附件，并通过网易 163 邮箱发送到公司邮箱。收件人在 Windows 上下载全部附件后，无需安装软件，只需执行邮件正文中的 `copy /b` 命令即可恢复原文件。

## 已实现功能

- 支持 `doc`、`docx`、`xls`、`xlsx`、`ppt`、`pptx`、`pdf`、`zip`、`rar`、`7z` 单文件发送。
- 默认约 7MB 的原始字节连续分片；大文件不解压、不转码、不重压缩。
- 发送前使用真实 MIME/Base64 邮件内容逐份预检；接近 9.9MB 上限时自动缩小分片。
- 生成固定排序附件名、邮件主题、Windows 合并命令和 SHA-256 校验值。
- 网易 163 SMTP（SSL）发送、每封间隔 2 秒、单封失败自动重试 3 次。
- 发送暂停、失败续发、应用关闭后的任务恢复，以及最近 30 天发送历史。
- 网易授权密码仅保存在 macOS 钥匙串或 Windows 凭据管理器；历史中不保存密码或原文件副本。

## 本地开发

需先安装 Node.js 22 或更高版本。

```bash
npm install
npm run dev
```

质量检查与生产构建：

```bash
npm run typecheck
npm test
npm run build
```

## 打包

在对应系统上执行：

```bash
# macOS
npm run pack:mac

# Windows
npm run pack:win
```

输出文件位于 `dist/`。macOS 内部试用包若未配置 Apple Developer ID，将显示为未签名应用；正式分发前请配置应用图标、Apple/Windows 代码签名与公证流程。

## 网易 163 邮箱准备

在网易邮箱网页版的设置中开启 SMTP/客户端服务，并生成“客户端授权密码”。应用设置页填写 `@163.com` 地址和该授权密码，不应填写网页登录密码。

## 收件端操作

1. 下载所有邮件附件到同一 Windows 文件夹。
2. 在资源管理器地址栏输入 `cmd` 并回车。
3. 复制任意一封邮件正文中的完整 `copy /b` 命令并执行。
4. 用邮件正文给出的 SHA-256 值核验合并后的文件。
